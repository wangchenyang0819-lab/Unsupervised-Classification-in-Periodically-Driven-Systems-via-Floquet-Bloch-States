# 2D-D Class

This repository contains MATLAB codes for unsupervised classification of two-dimensional periodically driven Floquet topological phases.

The classification is based on Floquet-Bloch eigenstates, a fidelity-like kernel construction, diffusion map dimensionality reduction, and visualization of the resulting phase clusters.

## Files

This project contains five main files:

- `fiveclass.m`: performs five-class phase classification by scanning the driving parameter J2.
- `sevenclass.m`: performs seven-class phase classification by scanning the driving parameter J3.
- `fig4.m`: generates the final combined figure using the saved classification results.
- `2DD1.mat`: result file generated by `fiveclass.m`.
- `2DD2.mat`: result file generated by `sevenclass.m`.

## Model

The model is a two-dimensional periodically driven two-band Floquet system.

Within one driving period, the Hamiltonian is divided into three piecewise constant steps:

    H1 = 3 * gamma1 * sigma_x
    H2 = 3 * gamma2 * sigma_y
    H3 = 3 * gamma3 * sigma_z

Each Hamiltonian acts for one third of the driving period T.

The full-period Floquet evolution operator is written as:

    U(T) = U3 * U2 * U1

The Floquet eigenstates are obtained by diagonalizing the full-period evolution operator. These eigenstates are then used to construct the kernel matrix for unsupervised phase classification.

## Description of Each File

### fiveclass.m

This script scans the parameter J2 and performs unsupervised classification of five Floquet topological phases.

In this script, J1 and J3 are fixed, while J2 is sampled over a selected parameter range.

The main procedure is:

1. Generate the J2 sample list.
2. Discretize the two-dimensional Brillouin zone.
3. Construct the Floquet evolution operator at each momentum point.
4. Diagonalize the Floquet operator to obtain Floquet eigenstates.
5. Compute the time-evolved Floquet wave functions.
6. Construct the fidelity-like kernel matrix.
7. Apply diffusion map dimensionality reduction.
8. Save the results to `results/2DD1.mat`.

The saved variables are:

- `eigvals`: eigenvalues of the normalized diffusion matrix.
- `score`: PCA projection of the diffusion coordinates.
- `V_sym`: eigenvectors of the symmetric normalized kernel matrix.

### fig4.m

This script loads the saved results from `2DD1.mat` and `2DD2.mat`, and then generates the final combined figure.

The figure includes:

- The five-class phase diagram obtained from the J2 scan.
- The seven-class phase diagram obtained from the J3 scan.
- Inset plots of the leading diffusion-map eigenvalues.
- Inset plots of the PCA-projected diffusion coordinates.
- Topological invariant curves for different parameter regions.

## Requirements

The codes are written in MATLAB.

Recommended MATLAB toolboxes:

- Parallel Computing Toolbox  
  This toolbox is required if `parfor` is used. If it is not available, replace `parfor` with `for`.

- Statistics and Machine Learning Toolbox  
  This toolbox is required for the `pca` function.

## How to Run

### Step 1: Create the results folder

Before running the classification scripts, create a folder named `results` in the project directory.

In MATLAB, run:

    mkdir results

### Step 2: Run the five-class classification

Run:

    fiveclass

This script generates:

    results/2DD1.mat

### Step 3: Run the seven-class classification

Run:

    sevenclass

This script generates:

    results/2DD2.mat

### Step 4: Generate the final figure

After both result files are generated, run:

    fig4

This script generates the final combined phase diagram.

## Output Data

The two classification scripts save the following variables:

- `eigvals`
- `score`
- `V_sym`

Their meanings are:

- `eigvals`: eigenvalues of the normalized diffusion matrix.
- `score`: PCA projection of the diffusion coordinates.
- `V_sym`: eigenvectors of the symmetric normalized kernel matrix.

## Notes

The kernel construction involves nested loops over momentum points, time points, and sample pairs. Therefore, the computation may be time-consuming when large values of `Nk`, `Nt`, or the number of samples are used.

To reduce the computational cost, one may decrease the following parameters:

- `Nk`
- `Nt`
- `m`

For faster computation, it is recommended to use MATLAB parallel computing with `parfor`.
