# Floquet Topological Classification for Class D (1D)

This folder contains MATLAB code for unsupervised clustering of Floquet topological phases in symmetry class D (1D). The method constructs a kernel from the flattened Floquet operator in momentum‑time space, then applies diffusion maps and hierarchical clustering to identify topologically distinct phases. Theoretical phase diagrams (based on analytic expressions) and Chern numbers are also provided for validation.

## Files

| File | Description |
|------|-------------|
| `blochd.m` | Generates Bloch vector components (xx, yy, zz) for a given set of parameters (e.g., for a representative phase). Saves the data as `kernelD151.mat` (or `kernelD0fu1.mat` if parameters are adjusted). |
| `dinvariantsdiffusion.m` | Main clustering script. Computes the theoretical phase diagram, builds the similarity kernel for a grid of samples, performs diffusion map + hierarchical clustering, and saves the result as `D.mat`. Also produces PCA plots. |
| `fig3.m` | (Not provided here) Plotting script that reads `D.mat`, `kernelD0fu1.mat`, and `kernelD151.mat` to produce Fig. 3 in the paper. |

## Data files

- `D.mat` – Generated by `dinvariantsdiffusion.m`. Contains:
  - `e`: diffusion map eigenvalues (used for spectral gap analysis)
  - `score`: first few principal components from PCA
  - `tpp`: theoretical phase label matrix (1–4) over the parameter grid
  - `B`: 2×`grid_size²` array of parameters (J₁, J₂) in units of π/T
  - `f`: cluster assignment matrix for plotting
  - `TT`: cluster indices for each sample
- `kernelD151.mat` – Generated by `blochd.m` for a specific parameter set (e.g., J₁=1.5π/T, J₂=1π/T). Contains `xx`, `yy`, `zz` (Bloch vector components in (k, t) space).
- `kernelD0fu1.mat` – (Optional) Similar file generated with different parameters (e.g., J₁=0.5π/T, J₂=0.3π/T) for a different topological phase.

## Requirements

- MATLAB (R2020b or later recommended)
- No special toolboxes required (the code uses basic functions only)

## Usage

### 1. Generate the theoretical phase diagram and perform clustering

Run `dinvariantsdiffusion.m` to:
- Compute the analytic phase boundaries based on trigonometric functions (4 distinct phases, labelled 1–4).
- Set up a 16×16 grid of parameters (J₁, J₂) within a square region.
- For each sample, construct the Floquet effective Hamiltonian (including up to 7th harmonics), compute the flattened Floquet operator, and evaluate the similarity kernel.
- Perform diffusion map and hierarchical clustering (number of clusters determined from eigenvalues close to 1).
- Save the results to `D.mat`.

```matlab
>> dinvariantsdiffusion
The script will produce two figures:

The theoretical phase diagram (colored surface).

The clustering result (colored/scattered points in parameter space).

2. Generate Bloch vector data for a specific phase
Edit blochd.m to set the coupling coefficients A(1,1) (J₁) and A(2,1) (J₂) to values corresponding to a topological phase of interest (e.g., J₁=1.5π/T, J₂=1π/T for the 0‑π phase). Then run the script to compute the time‑evolved Floquet‑Bloch states and the Bloch vector components (xx, yy, zz) in (k, t) space. The script also calculates and displays the Chern number.

matlab
>> blochd
The script saves the result as kernelD151.mat. To generate kernelD0fu1.mat for a different phase, change the coupling parameters (e.g., J₁=0.5π/T, J₂=0.3π/T) and save under a different filename (modify the save command accordingly).

3. Reproduce the paper figures
Run fig3.m (not provided in this folder) to load D.mat, kernelD0fu1.mat, and kernelD151.mat and produce the phase diagram, eigenvalue plot, PCA embedding, and Bloch vector visualizations as shown in the manuscript.

Output details
D.mat

e: eigenvalues of the diffusion map. The number of eigenvalues close to 1 (should be 8 for class D) indicates the embedding dimension.

score: PCA coordinates of the first principal components.

tpp: 400×400 matrix of theoretical phase labels (1 to 4) for the analytic phase diagram.

B: 2×256 array of (J₁, J₂) values in units of π/T.

f: 16×16 matrix of cluster indices (1 to 8) for each sample point.

TT: 256×1 vector of cluster assignments for hierarchical clustering.

kernelD151.mat (and kernelD0fu1.mat)

xx, yy, zz: N×N matrices of the Bloch vector components at each (k, t) grid point. Used to visualise the orientation of the Floquet‑Bloch state on the (k, t) torus.

Notes
The theoretical phase diagram in dinvariantsdiffusion.m is obtained analytically from sign(cos((x+y)π/4) * cos((x-y)π/4)) and sign(sin((x+y)π/4) * sin((x-y)π/4)). This provides a clean validation of the unsupervised clustering results.

The kernel constant epsilon (0.03) is chosen to be small enough to resolve topological differences without over‑splitting identical phases.

For other symmetry classes (A, AIII), please refer to the corresponding subfolders.
